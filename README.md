The project is run on `python version 3.10.7`

Requirements

`python 3.10.7`

- install from official website

virtualenv install

- `pip install virtualenv`

- `git clone "projectlink"`

### How to Run

create a virtual environment

by

- `python -m venv env_name`

activate the env

- `env_name\scripts\activate`

move to the folder where requirements.txt is

- `pip install -r requirements.txt`

run the file by

- `py main.py`
